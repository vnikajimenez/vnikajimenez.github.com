About

Veronica Jimenez

Web designer apprentice

San Francisco, California

I've always been inspired by visual elements. Growing up in a culture full of color and mystery like Mexico is one of the things I've always been thankful for.
When I finished college I've had the opportunity to travel for a couple of years, where I experienced new cultures, shapes and languages and this inspired me to do some photography.

So one thing took me to the another, and my passion for telling histories told me I needed to ge into web design.

Last year I started taking design, Adobe Suite, user experience at San Francisco State University. I've also been learning Html, Css and Javascript with General Assembly at the Front End Web Development intensive program.
Its so refreshing for me to learn about the web world and how it works.

Im looking for a place where I can put my new skills to work in real life and hopefully in some time soon I will be able to call myself a unicorn and designer
